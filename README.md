## Getting Started

### Prerequisites

- A modern web browser
- Git (for local development)

### Run Locally

```bash
# Clone the repository
git clone https://github.com/sreenathsjk/srichaithanyaschool.git

# Navigate into the project folder
cd srichaithanyaschool

# Open index.html in your browser
open index.html
```

No build step or npm install required — it's a static site.

---

## Deployment

This site is deployed automatically via **GitHub Pages** from the `main` branch.

Any push to `main` will reflect live on the site within a few minutes.

To enable GitHub Pages for a forked copy:
1. Go to **Settings → Pages**
2. Set source to **Deploy from branch → main → / (root)**
3. Save and wait for deployment
